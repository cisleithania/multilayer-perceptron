# Классификация данных электроэнцефалограммы с помощью многослойного перцептрона.

Программное обеспечение, может с помощью многослойного перцептрона,
классифицировать экспериментально полученные данные с ЭЭГ.
Нейронная сеть обучается методом обратного распространения ошибки и 
реализована на языке С++, с помощью библиотеки FANN 2.2.0.
